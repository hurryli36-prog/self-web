# 自我关怀泡泡 Snapshot Demo

一个极简的前端原型，演示「自我关怀泡泡 App」在快照层围绕一颗液态玻璃球展开的体验：

- **单一核心 · 球就是界面**：首页只剩一行问候 + 情绪核球，背景是 visionOS 风格的蓝紫渐变、模糊光斑与轻微视差，其他信息全部沉入球体交互。
- **玻璃质地的情绪核**：半透明材质、背后模糊、内外高光与缓慢呼吸动画，让球看起来像真的液态玻璃；轻点即可暂停/恢复流动。
- **液态特效被层层叠加**：新增的漂浮光斑、粒子噪点、玻璃雾和 SVG 液态位移滤镜，让球与背景互相折射、呈现真玻璃的厚度与流动感。
- **长按呼吸出的花瓣菜单**：长按 0.6 秒，球周围浮出 5 颗半透明 pill，像花瓣或行星环绕；拇指向某个方向一滑即选中“放松一下 / 推进一步 / 整理情绪 / 来点灵感 / 只是看看”。
- **即时回应与色调联动**：选择花瓣后，球根据该意向调节色温、亮度、流速，下方玻璃胶囊也回声式地给出中文反馈与两个小行动建议。
- **长按波纹和行星视差**：长按呼出菜单时球体会放大、喷出同心波纹，浮动光斑与背景渐变同步轻微偏移，暗示可以继续拖动、探索历史。
- **纯粹的液态舞台**：多余的卡片、历史条都被拿掉，只留下巨大的玻璃球、极简文案和两个底部出口，让画面更贴近 Apple visionOS 26 的 glassmorphism。 

打开 `index.html` 即可体验，无需构建步骤。
